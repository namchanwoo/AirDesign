## 2. 캐릭터 네러티브

### 2.1 배경 스토리

#### 침묵의 마을 - 희생양

검은 재앙이 퍼지기 시작했을 무렵, 대륙 경계에 있던 작은 마을이 있었다. 검은 안개가 스며든 것은 어느 날 밤. 사람들이 잠든 사이였다.

그 안개 속에는 **무언가**가 있었다.

시공간이 비틀어진 검은 재앙 안에서 태어난 존재. 젊음, 기억, 희망... **추상적인 것들을 뺏어가거나 먹는** 기괴한 존재였다. 그것은 침묵의 마을을 **희생양**으로 삼았다.

#### 빼앗긴 성장

에르나가 그날 밤 눈을 떴을 때, 그녀의 몸은 열한 살이었다.

하지만 아침이 밝았을 때, 그녀는 여전히 열한 살이었다. 1년이 지나도, 10년이 지나도. 그녀의 몸은 **성장을 멈췄다**.

미지의 존재에게 **"성장"을 빼앗겼기** 때문이다.

저주가 아니었다. 약탈당한 것이다. 성장할 수 있는 가능성, 어른이 될 수 있는 미래, 몸이 자라는 시간 - 그 모든 것을.

```
"저주라면 풀 수라도 있겠지. 하지만 빼앗긴 건... 되돌릴 수 없어."
"그래도 괜찮아. 작은 몸에도 익숙해졌거든. 수십 년이나 지났으니까."
```

#### 망자들 - 그리고 유일한 생존자

더 심하게 빼앗긴 자들은 **"망자"**가 되었다.

자신이 누구인지 잊어버린 자들. 이름도, 얼굴도, 사랑하는 사람도 기억하지 못하는 자들. 그들은 텅 빈 눈으로 마을을 배회하다 어느 날 사라졌다. 검은 안개 속으로 돌아간 것인지, 아니면 어딘가에서 쓰러진 것인지... 아무도 모른다.

에르나의 부모도. 형제도. 친구들도. 모두.

에르나만이 **유일한 생존자**였다.

성장을 빼앗겼지만, 그녀의 기억은 대부분 남았다. 왜 자신만 살아남았는지, 에르나는 알지 못한다. 다만 **마을 사람들이 사라지는 것을 모두 지켜봤다**는 사실만이 선명하다.

```
"다들 점점 나를 못 알아봤어. 부모님조차도."
"마지막엔 자기 이름도 기억 못 하더라. 그리고... 사라졌어."
"그날 밤, 나는 혼자였어. 완전히."
```

#### 잃어버린 조각들

에르나 역시 완전히 무사한 것은 아니다. 기억의 일부가 **빼앗겼다**.

어릴 적 추억의 일부, 부모님과 나눈 대화들, 친구들과 뛰놀던 어느 여름날 - 그런 것들이 안개처럼 흐릿하다. 분명 있었던 기억인데, 손을 뻗으면 스르르 사라진다.

하지만 기이하게도, **검을 휘두르는 기억은 남았다**.

```
"기억은 흐릿해도, 몸은 기억해. 이상하지? 수십 년 동안 훈련했거든."
"잊어버린 것과 남은 것... 무슨 기준인지는 모르겠어."
```

#### 구원의 사명

에르나가 검은 재앙에 들어가는 이유는 단순하다.

**망자들의 시체를 찾기 위해**. 그리고 **부활시키기 위해**.

루의 신전은 시체만 있으면 부활이 가능하다. 시체 상태가 좋으면 기억도 돌아온다. 에르나는 믿고 있다 - 마을 사람들을 찾아내고, 부활시키면, 다시 예전처럼 될 수 있을 거라고.

그게 불가능하더라도.

적어도 **시체라도 찾아서 장례는 치러줘야 하지 않겠는가**.

```
"다들 어딘가에 있을 거야. 검은 안개 속 어딘가에."
"찾아낼 거야. 전부. 그리고 돌려놓을 거야."
"...돌아오지 못하더라도, 적어도 혼자 두진 않을 거야."
```

#### 죄책감과 쾌활함

왜 자신만 살아남았는가.

에르나는 그 질문에 답을 찾지 못했다. 그래서 **웃기로 했다**. 울어봤자 아무것도 돌아오지 않으니까. 슬퍼해 봤자 힘만 빠지니까.

수십 년간 혼자 살아남은 그녀가 찾은 답은 단순했다.

**살아 있는 동안은, 웃으면서 할 수 있는 걸 하자.**

```
"울어봤자 키도 안 크고 주름도 안 늘어! 그럼 뭐하러 울어?"
"...가끔은 울고 싶을 때도 있어. 근데 참아. 울 시간에 한 발자국이라도 더 가야 하니까."
```

### 2.2 성격과 특성

#### 쾌활하고 밝음

```
"울어봤자 키도 안 크고 주름도 안 늘어! 그럼 뭐하러 울어?"
```

모든 것을 잃었음에도 에르나는 밝다. 아니, 밝아지기로 **선택했다**. 슬픔에 빠져 있기엔 해야 할 일이 너무 많았다. 찾아야 할 사람들이 있으니까.

**쾌활함은 갑옷**이다. 수십 년간 혼자 살아남은 그녀가 무너지지 않기 위해 만든 방어막.

#### 색드립과 성인 유머

```
"우리 마을에서 내가 제일 쭉쭉빵빵이야! ...뭐? 안 믿어? 이 몸에 수십 년 치 경험이 담겨 있다고!"
"큰 거 좋아해. 무기도, 음식도, 남자도... 아, 물론 농담이야. 반은."
```

수십 년을 산 정신으로 11세의 입을 통해 나오는 말들은 주변 사람들을 당혹스럽게 만든다. 에르나 본인은 그 반응을 즐긴다. 웃음은 침묵보다 낫다.

#### "큰 것"에 대한 취향

작은 몸에 갇힌 반동인지, 에르나는 **크고 웅장한 것**에 끌린다. 거대한 무기, 넓은 하늘, 큰 음식, 큰 꿈.

**자라지 못한 것에 대한 보상심리**일지도 모른다. 그래서 그녀의 무기도 비상식적으로 크다.

#### 노련한 판단력

외형과 달리, 에르나의 전투 감각은 수십 년의 경험에서 나온다. 언제 돌진해야 하는지, 언제 방어해야 하는지 - 그녀는 그 누구보다 잘 안다.

많은 전투를 보았고, 많은 죽음을 목격했기 때문이다. **그리고 많은 사람들이 사라지는 것도**.

#### 죄책감

평소에는 드러내지 않지만, 에르나는 **생존자의 죄책감**을 안고 있다. 왜 자신만 살아남았는가. 왜 다른 사람들은 망자가 되었는데 자신은 아닌가.

그 질문에 답하기 위해 그녀는 검은 재앙 속으로 들어간다.

## 4. 핵심 시스템: 기억 스택 (Memory Stack)

### 4.0 기억 스택 개요

> *"빼앗긴 기억이라도, 몸은 기억해."*

에르나의 핵심 메커니즘. 미지의 존재에게 기억의 일부를 빼앗겼지만, **전투 중 행동할 때마다 몸에 새겨진 경험이 되살아난다**.

**네러티브 연결**:
- 에르나가 빼앗긴 것 중 하나는 "기억"
- 하지만 **몸에 새겨진 기술, 전투 경험은 빼앗을 수 없었다**
- 전투 중 쌓이는 "기억 스택"은 **빼앗긴 기억의 조각들이 일시적으로 되살아나는 것**
- 5스택 발동 = 수십 년간의 경험이 한순간 폭발하는 순간

```
┌─────────────────────────────────────────────────────────────────┐
│                      [기억 스택 시스템]                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ■ 획득 조건 (형태별 차별화)                                     │
│                                                                 │
│    [대검 형태]                                                  │
│    → 공격 시 기억 스택 +1                                       │
│    → 적 처치 시 기억 스택 +2                                    │
│                                                                 │
│    [방패 형태]                                                  │
│    → 피격 시 기억 스택 +1                                       │
│    → 피격 후 HP 30% 이하면 기억 스택 +2                         │
│                                                                 │
│  ■ 최대 스택: 5                                                  │
│                                                                 │
│  ■ 스택 유지                                                     │
│                                                                 │
│    ★ 한번 쌓인 스택은 사용 전까지 감소하지 않음 ★               │
│    → 턴 경과로 감소 X                                           │
│    → 형태 전환으로 감소 X                                        │
│    → 사망 직전까지 유지                                          │
│                                                                 │
│  ■ 스택 효과 (5스택일 때 자동 발동)                              │
│                                                                 │
│    ★ 5스택 도달 시, 다음 행동에서 자동으로 효과 발동 + 소모 ★    │
│                                                                 │
│    [공격 행동 시] → 50% 방어력 관통                             │
│    [방어 행동 시] → 50% 피해 감소                               │
│                                                                 │
│    ※ 유저가 스택 사용 여부를 선택할 수 없음                      │
│    ※ 5스택이 되면 자동으로 발동되어 0으로 초기화                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**설계 의도**:
- 스택 관리의 복잡성 제거 → 직관적인 "채우면 발동" 시스템
- **스택이 감소하지 않음** → 안정적인 스택 축적, 형태 전환에 페널티 없음
- 플레이어는 **5스택 발동 타이밍을 공격/방어 중 어디에 쓸지** 선택
- 5스택 = 강력한 보너스, 어떤 행동에 발동시킬지가 전략의 핵심

### 9.2 추천 파티 조합

| **에르나 + 힐러** | 에르나의 피격을 힐러가 회복, 지속 전투 가능 |
| **에르나 + 버프 서포터** | 에르나의 공격력/방어력 버프로 역할 극대화 |

| 상황 | 대응 |

| **지속 피해 (DoT)** | 방패 형태로 스택 축적 (피격당 +1), 5스택 피해 감소 활용 |

| **외형 나이** | 30대 중반 |

### 1.2 컨셉 키워드

```
갭 모에 | 온화한 고위사제 외형 + 타락한 과거 | 치유자 + 한때 파괴자였던 자
술과 담배에 찌든 사제 + 여전히 경건한 신앙심 | 신비롭게 나타나 돕고 사라지는 자 + 내면의 죄책감
고통 전이 | 치유 = 적 약화 | 아군의 상처를 적에게
외우주 오염 | 정신 침식 | 한때 하수인이었던 자 | 속죄의 여정
검은 재앙의 방랑자 | 희생자들의 고통을 품은 자
```

### 1.3 핵심 플레이 경험

**"치유의 대가는 적이 치른다"**

- 아군을 치유할수록 적이 약해지는 독특한 힐러 경험
- 아군의 상처를 "고통 스택"으로 흡수하고, 적에게 전이하여 디버프 부여
- 직접적인 공격력은 낮지만, 아군 지원 + 적 약화의 이중 역할로 전투 흐름 장악
- "나는 치유한다. 그리고 적은 무너진다."

## 2. 캐릭터 네러티브

### 2.1 배경 스토리

#### 재생의 고위 사제 - 생명을 살리는 자

카이렌은 재생과 치유를 관장하는 신을 섬기던 **고위 사제**였다. 신전에서 가장 뛰어난 치유사로, 그의 손길이 닿으면 어떤 상처도 아물었다. 수많은 병자와 부상자들이 그의 신전을 찾았고, 그는 한 사람도 거부하지 않았다.

```
"신의 은총은 모두에게 평등합니다. 손을 내밀 수 있는 한, 저는 치유할 것입니다."
"카이렌 대사제님, 오늘도 쉬지 않으시렵니까..."
"고통받는 이가 있는데 어찌 쉴 수 있겠습니까."
```

#### 검은 재앙에서의 조우 - 타락의 시작

어느 날, 검은 재앙이 확장하며 그의 신전 인근 마을까지 삼켰다. 카이렌은 생존자를 구하기 위해 **혼자 검은 재앙 속으로 들어갔다**.

그곳에서 그는 **외우주의 존재**와 마주했다.

인간의 정신으로는 이해할 수 없는 것. 존재 자체가 인지재해인 것. 그것과 눈이 마주친 순간, 카이렌의 **정신은 오염**되었다.

```
"그것은... 형체가 없었습니다. 아니, 형체가 너무 많았습니다."
"목소리가... 제 안에서 들렸습니다. 그리고 저는..."
"...저는 더 이상 제가 아니게 되었습니다."
```

#### 하수인의 시절 - 용서받지 못할 죄

오염된 카이렌은 **외우주 존재의 하수인**이 되었다. 그의 치유의 손은 파괴의 도구가 되었다.

검은 재앙 안에서, 그는 수많은 결사대원과 모험가들을 **죽였다**. 구하러 온 자들을, 도망치려는 자들을. 치유사의 손이 가장 잔혹한 손이 되었다.

얼마나 많은 이들이 그의 손에 죽었는지, 정확한 숫자는 알 수 없다. 하지만 그 **고통의 기억**은 모두 그의 정신에 새겨졌다.

```
"도망쳐! 저 사제... 저 사제가...!"
"살려... 살려주..."
"왜... 왜 이러시는 겁니까, 카이렌 대사제님..."
```

#### 정신을 차린 순간 - 자살

어느 날, 카이렌은 **일순간 정신을 차렸다**.

주변에는 시체가 쌓여 있었다. 그의 손에는 피가 묻어 있었다. 그리고 그는 **자신이 무엇을 했는지** 기억해냈다.

다시 오염되기 전에, 다시 하수인이 되기 전에 - 그는 **스스로 목숨을 끊었다**.

```
"다시는... 다시는 안 돼..."
"이 손으로 더 이상은..."
```

#### 루의 신전에서 - 부활과 예언

루의 신전에서 부활했을 때, 카이렌은 **또다시 죽으려 했다**.

하지만 신전의 사제들이 그를 막았다. 그리고 **예언**을 전했다.

```
"당신이 모시던 신의 뜻으로 부활하셨습니다, 카이렌 대사제님."
"...그럴 리가. 저는... 저는 용서받을 수 없는..."
"당신은 유일한 그 곳의 고위사제입니다. 신전은 무너졌지만, 신앙은 살아있습니다."
"...신앙의 힘이... 아직 남아있다고요?"
"믿음의 증거입니다. 당신은 아직 버림받지 않았습니다."
```

그의 치유 능력은 **그대로였다**. 타락하고, 죄를 저지르고, 죽음을 거쳤음에도 - 신앙의 힘은 그에게서 떠나지 않았다.

#### 속죄의 여정 - 검은 재앙의 방랑자

카이렌은 **혼자 검은 재앙 속으로 들어가기 시작했다**.

결사대와 모험가들이 위험에 처할 때, **어디선가 나타나 치유하고** 유유히 사라지는 신비한 인물. 검은 재앙의 방랑자. 누구도 그의 이름을 모르고, 누구도 그를 붙잡지 못한다.

그가 구한 사람들은 그를 은인이라 부르지만, 그 자신은 **속죄**라 부른다.

```
"왜 구해주신 건가요? 대가도 없이?"
"...대가는 이미 받았습니다. 아주 오래전에."
"?"
"신경 쓰지 마세요. 그냥... 살아 돌아가세요."
```

#### 돌아오는 기억 - 술과 담배

하지만 밤이 되면, **기억이 돌아온다**.

하수인으로서 저지른 죄악들. 희생자들의 얼굴, 비명, 마지막 말. 그 고통의 기억이 그의 정신을 갉아먹는다.

카이렌은 그 고통을 견디기 위해 **술과 담배에 의존**하게 되었다.

```
"대사제님... 이 늦은 시간에 또 술을...?"
"...잠이 오지 않아서요."
"악몽이신가요?"
"악몽이었으면 좋겠어요. 악몽은... 깨어나면 끝나니까요."
```

#### 주인공과의 만남 - 사명의 발견

주인공(아이온)을 처음 만났을 때, 카이렌은 **묘한 느낌**을 받았다.

여명의 권능. 태초의 신. 검은 재앙을 끝낼 수 있는 존재.

그리고 직감했다. 이것이 자신이 **다시 살아난 이유**라고. 이것이 자신의 **사명**이라고.

```
"...당신이군요. 소문으로 듣던."
"소문이요?"
"여명의 권능. 검은 재앙을 물리칠 수 있는 자. ...함께 가도 될까요?"
"갑작스럽군요. 이유가 있습니까?"
"...제가 섬기던 신이 저를 이 순간을 위해 살려두신 것 같아서요."
```

#### 목표 - 복수와 속죄

카이렌의 목표는 명확하다.

첫째, 자신의 정신을 오염시킨 **외우주 존재를 멸하는 것**.
둘째, 자신이 하수인으로서 저지른 **죄의 값을 치르는 것**.

그가 구한 사람의 수가 그가 죽인 사람의 수를 넘을 때까지. 아니, 영원히 넘을 수 없을지도 모르지만 - 그래도 멈출 수 없다.

```
"당신은 충분히 속죄했어요, 카이렌."
"...아니에요. 충분한 속죄 같은 건 없어요."
"그럼 언제까지?"
"그 존재가 사라질 때까지. 아니, 그 이후에도. 영원히."
```

### 2.2 성격과 특성

#### 온화하지만 지친 모습

```
"걱정 마세요. 제가 치료해 드릴게요."
```

카이렌은 여전히 부드러운 미소를 짓는다. 목소리는 차분하고, 손길은 따뜻하다. 환자를 대할 때 그는 완벽한 치유사다.

하지만 그 온화함 뒤에는 **깊은 피로**가 있다. 술과 담배 냄새가 살짝 묻어 있고, 눈 밑에는 잠 못 드는 밤의 흔적이 있다.

#### 내면의 죄책감

희생자들의 고통이 뒤엉킨 정신. 카이렌은 때때로 **기억에 잠식**당한다.

- 갑자기 멈추고 허공을 응시하거나
- 누군가의 비명이 들린 듯 고개를 돌리거나
- 손에 묻은 피를 씻는 듯한 동작을 반복하거나

```
"이 손으로... 이 손으로 얼마나 많은 사람을..."
"카이렌, 괜찮아요?"
"...네. 괜찮아요. 잠깐... 기억이..."
```

#### 경건함과 자기혐오의 공존

그는 여전히 **신앙심이 깊다**. 매일 아침 기도하고, 치유 전 축복을 읊조린다. 신앙의 힘이 남아있다는 것은, 신이 그를 아직 버리지 않았다는 의미다.

하지만 동시에 **자신을 용서하지 못한다**. 왜 신이 자신을 버리지 않는지 이해할 수 없다.

```
"신이시여, 왜 아직도 저를 쓰시는 겁니까..."
"신앙의 힘이 남아있다면, 그건 용서의 증거 아닌가요?"
"...용서받을 자격이 없는데도요."
```

#### 술과 담배 - 고통을 견디는 방법

밤마다 돌아오는 기억. 희생자들의 얼굴, 비명, 마지막 말. 그 고통을 견디기 위해 카이렌은 **술과 담배**에 의존한다.

고위 사제다웠던 깔끔한 모습은 사라졌다. 하지만 그것이 그를 버틸 수 있게 해준다.

```
"대사제가 술을 마시다니..."
"대사제였어요. 이제는 그냥... 죄인이에요."
"그래도 이렇게까지..."
"...아침이 되면 괜찮아질 거예요. 늘 그랬듯이."
```

#### "희생자들의 고통을 품은 자"

카이렌은 자신이 죽인 이들의 **고통의 기억**을 품고 있다. 하수인 시절, 치유의 힘이 뒤틀려 고통을 흡수하게 되었다. 그가 죽인 모든 이들의 마지막 순간이 그의 정신에 새겨져 있다.

```
"가끔... 그들의 목소리가 들려요. 제가 죽인 사람들."
"환청인가요?"
"...기억이에요. 제 안에 있는. 영원히 지워지지 않는."
```

### 2.3 관계

| 대상 | 관계 |
|------|------|
| **희생자들 (망자)** | 그의 안에 있는 존재들. 그가 하수인 시절 죽인 이들의 고통과 기억. 영원히 속죄해야 할 죄의 무게. |
| **주인공 (아이온)** | 사명의 대상. 여명의 권능이 검은 재앙을 끝낼 수 있다면, 그것이 자신이 다시 살아난 이유. 아이온을 지키는 것이 자신의 속죄. |
| **신 루 (Lux)** | 부활시켜 준 존재. 그리고 자신이 섬기던 신이 루의 신전과 연결되어 있을 가능성. 예언의 의미를 아직 완전히 이해하지 못함. |
| **재생의 신 (옛 신앙)** | 카이렌이 섬기던 신. 타락 후에도 신앙의 힘을 거두지 않은 존재. 왜 버리지 않는지 이해할 수 없지만, 그 믿음에 보답하려 함. |
| **에르나** | 같은 검은 재앙과 싸우는 동료. 에르나도 무거운 짐을 지고 있음을 느낀다. 서로의 고통을 이해하는 사이. |
| **외우주 존재** | 자신을 타락시킨 원흉. 반드시 멸해야 할 대상. 복수와 속죄의 최종 목표. |
| **검은 재앙** | 속죄의 장소. 그곳에서 고통받는 이들을 구하는 것이 자신의 죄를 갚는 방법. |

## 3. 무기 시스템

### 3.1 개요

카이렌은 **무기를 사용하지 않는다**. 그의 힘은 손에서 나온다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    [맨손 치유술 / 정신 간섭]                       │
│                                                                 │
│      ┌──────────────────┐                                       │
│      │   치유의 손길    │                                       │
│      │                  │                                       │
│      │  아군 상처 흡수  │                                       │
│      │  적에게 고통 전이│                                       │
│      └──────────────────┘                                       │
│                                                                 │
│      시각적 특징: 손에서 검푸른 안개가 피어오름                   │
│      치유 시 온화한 빛, 전이 시 불길한 빛                         │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 시각적 표현

| 상황 | 시각적 효과 |
|------|------------|
| **치유 시** | 손에서 온화한 금빛이 퍼져나감, 아군을 감싸는 따뜻한 오라 |
| **고통 흡수 시** | 아군에게서 검푸른 실이 카이렌에게로 흘러들어옴 |
| **고통 전이 시** | 카이렌의 손에서 불길한 검붉은 파동이 적에게 뻗어나감 |
| **5스택 발동 시** | 카이렌 주변에 희생자들의 희미한 얼굴이 순간적으로 나타났다 사라짐 |

## 4. 핵심 시스템: 고통 스택 (Pain Stack)

### 4.0 고통 스택 개요

> *"치유할수록, 나는 더 많은 고통을 품게 된다. 하지만 이제 그것은 무기가 되었다."*

카이렌의 핵심 메커니즘. 아군을 치유하거나 아군이 피해를 입을 때 **고통 스택**을 획득하고, 이를 **적에게 전이**하여 디버프를 부여한다.

**네러티브 연결**:
- 외우주의 힘에 오염된 카이렌은 고통을 "흡수"하고 "저장"할 수 있다
- 하수인 시절 죽인 희생자들의 고통이 그의 정신에 새겨져 있다
- 5스택 발동 = 축적된 고통이 폭발하여 적에게 전이되는 순간

```
┌─────────────────────────────────────────────────────────────────┐

### 9.3 포지셔닝 가이드

```
[권장 위치 - 후방 중앙]

아군 진영 (3×3 기준)
열1   열2   열3

├────┼────┼────┤

## 1. 시스템 개요

### 1.1 설계 의도

노드맵 이벤트 시스템은 **전투 외 구간에서 플레이어에게 의미 있는 선택과 서사적 경험을 제공**하는 것을 목표로 한다.

#### 핵심 설계 원칙

| 원칙 | 설명 | 참조 게임 |
|------|------|----------|
| **선택의 무게** | 모든 선택에는 명확한 트레이드오프 존재 | Slay the Spire |
| **캐릭터 서사 연동** | 이벤트가 파티원 관계와 기억 회복에 영향 | Hades |
| **외우주 침식** | 검은 재앙의 정신 오염이 이벤트에도 영향 | Darkest Dungeon |
| **런 다양성** | 동일 이벤트도 상황에 따라 다른 결과 | 로그라이트 공통 |

#### 세계관 연결

| 세계관 요소 | 이벤트 시스템 연결 |
|------------|-------------------|
| **검은 재앙** | 던전 내 이벤트의 위험도와 변형 이벤트 트리거 |
| **주인공(아우로라)의 기억 상실** | 스토리 이벤트를 통한 과거 회복 |
| **원정대원 부활** | 파티원 개인 이벤트, 신뢰도 시스템 |
| **루의 신전** | 안전지대 이벤트, 부활/회복 서비스 |
| **외우주 존재** | 적 조우, 인지재해 이벤트 |

### 1.2 이벤트 발생 구조

```
[노드 진입]
    │
    ▼
[노드 타입 확인]
    │
    ├─ ❓ 이벤트 노드 → 랜덤 이벤트 풀에서 추첨
    ├─ 🏕️ 휴식 노드 → 휴식 이벤트 + 조건부 특수 이벤트
    ├─ 🏪 상점 노드 → 상점 이벤트 + 조건부 특수 이벤트
    ├─ 💎 보물 노드 → 보상 이벤트 + 조건부 저주/축복 이벤트
    ├─ ⛪ 신전 노드 → 루의 신전 이벤트 (회복/부활/정보)
    └─ 📜 스토리 노드 → 고정 스토리 이벤트 (기억 회복)
    │
    ▼
[이벤트 선택지 제시]
    │
    ▼
[플레이어 선택]
    │
    ▼
[결과 적용 + 후속 효과]
```

### 1.3 이벤트 발생 조건 시스템

이벤트는 다양한 조건에 따라 출현 여부와 선택지가 결정된다.

| 조건 타입 | 예시 | 영향 |
|----------|------|------|
| **파티 구성** | 특정 원정대원 포함 시 | 특정 이벤트 해금, 추가 선택지 |
| **신뢰도** | 특정 캐릭터 신뢰도 40 이상 | 개인 이벤트 해금 |
| **정신 오염도** | 오염도 60 이상 | 변형 이벤트, 외우주 선택지 출현 |
| **보유 자원** | 파편 10개 이상 | 특수 거래 선택지 |
| **진행도** | 3장 이후 | 고급 이벤트 풀 해금 |
| **아우로라 기억** | 특정 기억 파편 획득 후 | 과거 관련 이벤트 해금 |
| **이전 선택** | 특정 이벤트에서 A 선택 | 후속 이벤트 분기 |

## 2. 이벤트 분류 체계

### 2.1 대분류

| 분류 | 설명 | 주요 노드 | 빈도 |
|------|------|----------|------|
| **캐릭터 성장** | 능력치, 인텐트, 카드 슬롯 확장 | 이벤트, 보물 | 15% |
| **캐릭터 스토리** | 개인 서사, 신뢰도 연동, 기억 회복 | 이벤트, 스토리 | 20% |
| **비전투** | 자원 관리, 탐색, 도박 | 이벤트 | 40% |
| **안전지대** | 상점, 휴식, 대장간, 신전 | 휴식, 상점, 신전 | 25% |

### 2.2 희귀도 체계

| 희귀도 | 출현 확률 | 보상 수준 | 리스크 |
|--------|----------|----------|--------|
| **일반** | 60% | 낮음 | 낮음 |
| **희귀** | 30% | 중간 | 중간 |
| **전설** | 8% | 높음 | 높음 |
| **고유** | 2% | 특수 | 가변 |

### 2.3 정신 오염도 변형 시스템

정신 오염도가 특정 임계점을 넘으면 이벤트가 **외우주 침식(Corrupted)** 버전으로 전환된다.

| 오염도 단계 | 변형 확률 | 변형 효과 |
|------------|----------|----------|
| 0-20 (안정) | 0% | 정상 이벤트 |
| 21-40 (동요) | 10% | 선택지 텍스트 왜곡 |
| 41-60 (불안) | 25% | 숨겨진 선택지 출현 (외우주 관련) |
| 61-80 (침식) | 50% | 결과 반전 가능성, 현현자 영향 |
| 81-100 (붕괴) | 75% | 완전 변형 이벤트, 심연자 암시 |

## 3. 캐릭터 성장 이벤트

### 3.1 스탯 성장 이벤트

#### 3.1.1 설계 원칙

- **선택적 성장**: 한 능력을 올리면 다른 것을 희생
- **캐릭터 특화**: 각 캐릭터의 역할을 강화하는 방향
- **아우로라의 권능**: 주인공 권능(여명)으로 파티원 잠재력 개방

#### 3.1.2 스탯 성장 밸런스 테이블

| 스탯 | 일반 이벤트 | 희귀 이벤트 | 전설 이벤트 |
|------|------------|------------|------------|
| 최대 HP | +2~3 | +4~6 | +8~10 |
| 공격력 | +1 | +2 | +3~4 |
| 방어력 | +1 | +2 | +3 |
| 스피드 | +1 | +1~2 | +2~3 |

> **이벤트 예시**: 노드맵_이벤트_컨텐츠_기획서.md 3.1절 참조

### 3.2 인텐트 해금 이벤트

#### 3.2.1 설계 원칙

인텐트 해금은 **신뢰도 시스템과 연동**되며, 이벤트를 통해 추가 해금 경로를 제공한다.

| 해금 경로 | 조건 | 설명 |
|----------|------|------|
| 신뢰도 경로 | 신뢰도 30/80 도달 | 기본 해금 경로 |
| 이벤트 경로 | 특수 이벤트 완료 | 신뢰도 요구치 -20 감소 |
| 개인 퀘스트 경로 | 개인 퀘스트 완료 | 3번째 인텐트 해금 |
| 기억 회복 경로 | 아우로라 관련 기억 이벤트 | 원정대원 전용 특수 인텐트 |

> **이벤트 예시**: 노드맵_이벤트_컨텐츠_기획서.md 3.2절 참조

### 3.3 카드 슬롯 확장 이벤트

#### 3.3.1 시스템 개요

각 캐릭터는 **기본 카드 슬롯 3개**를 보유하며, 이벤트를 통해 **최대 5개**까지 확장 가능하다.

| 슬롯 | 상태 | 해금 방법 |
|------|------|----------|
| 1~3 | 기본 해금 | 게임 시작 시 |
| 4 | 잠김 | 성장 이벤트 또는 신뢰도 50 |
| 5 | 잠김 | 희귀 이벤트 또는 신뢰도 90 |

> **이벤트 예시**: 노드맵_이벤트_컨텐츠_기획서.md 3.3절 참조

### 3.4 특수 능력 획득 이벤트

#### 3.4.1 특수 능력 체계

특수 능력은 **패시브 형태**로 런 동안 지속되며, 캐릭터당 최대 **2개**까지 장착 가능하다.

| 등급 | 획득 방법 | 효과 강도 |
|------|----------|----------|
| 일반 | 이벤트, 보물 | 약 |
| 희귀 | 희귀 이벤트, 현현자 처치 | 중 |
| 전설 | 전설 이벤트, 심연자 처치 | 강 |

#### 3.4.2 특수 능력 예시 목록

| 능력명 | 등급 | 효과 | 획득 이벤트 |
|--------|------|------|------------|
| 그림자 걸음 | 일반 | 전투 시작 시 은신 1턴 | 어둠의 틈새 |
| 철벽 의지 | 일반 | 정신 오염도 증가량 -20% | 명상의 샘 |
| 외우주 저항 | 희귀 | 침투자 공격 피해 -30% | 현현자 처치 |
| 여명의 잔광 | 전설 | 치명타 회피 1회 (런당 1회) | 아우로라 기억 완전 회복 |

> **이벤트 예시**: 노드맵_이벤트_컨텐츠_기획서.md 3.4절 참조

## 4. 캐릭터 스토리/퀘스트 이벤트

### 4.1 개인 스토리 이벤트 체계

#### 4.1.1 설계 원칙

- **신뢰도 연동**: 신뢰도 단계에 따라 새로운 이벤트 해금
- **기억 회복 연동**: 원정대원의 경우 아우로라와의 과거 기억이 핵심
- **선택의 영향**: 플레이어 선택이 캐릭터 관계와 결말에 영향
- **반복 불가**: 한 번 본 스토리 이벤트는 동일 런에서 재등장하지 않음

#### 4.1.2 신뢰도 단계별 이벤트

| 신뢰도 단계 | 구간 | 해금 이벤트 | 내용 |
|------------|------|------------|------|
| 낯선 이 | 0-9 | 첫 만남 | 캐릭터 소개, 기본 성격 |
| 인연 | 10-29 | 과거의 단서 | 원정대 시절 또는 과거에 대한 암시 |
| 신뢰 | 30-49 | 기억 회상 | 아우로라와 함께한 과거 일부 공개 |
| 깊은 유대 | 50-79 | 핵심 비밀 | 캐릭터의 핵심 트라우마/비밀 |
| 운명 공동체 | 80-100 | 운명의 갈림길 | 캐릭터 결말 분기점 |

> **이벤트 예시**: 노드맵_이벤트_컨텐츠_기획서.md 4.1절 참조

### 4.4 파티원 관계 이벤트

#### 4.4.1 관계 시스템

파티원 간에도 **관계도**가 존재하며, 특정 조합의 파티원이 함께할 때 **특수 이벤트**가 발생한다.

| 관계 타입 | 설명 | 효과 |
|----------|------|------|
| 동료 | 기본 관계 | 없음 |
| 전우 | 함께 전투 10회+ (원정대원 간) | 협동 이벤트 해금, 원정대 기억 공유 |
| 라이벌 | 특정 선택 충돌 | 경쟁 이벤트 해금 |
| 갈등 | 관계 악화 | 갈등 해소 이벤트 필요 |

## 5. 비전투 이벤트

### 5.1 파티 상호작용 이벤트

#### 5.1.1 설계 의도

전투 외 시간에 **파티의 인간적인 면모**를 보여주고, 플레이어와의 유대감을 형성한다.
원정대원의 경우 **20년 전 기억**과 **현재의 관계**가 교차한다.

#### 5.1.2 이벤트 유형

| 유형 | 트리거 | 효과 |
|------|--------|------|
| 식사 이벤트 | 휴식 노드 | 사기/신뢰도 소폭 상승 |
| 기억 공유 | 원정대원 + 아우로라 조합 | 기억 파편 힌트, 신뢰도 상승 |
| 논쟁 이벤트 | 특정 선택 후 | 관계 변화 |
| 위기 대처 | 랜덤 | 협동/갈등 |

## 9. 파티원 관련 희귀 이벤트

파티원 10명 각각의 개인 서브스토리를 5단계에 걸쳐 펼치는 희귀 이벤트 시스템이다. 지역별로 특화된 이벤트가 발생하며, 파티원의 과거와 현재, 그리고 아우로라와의 관계를 심화한다.

### 9.1 파티원 희귀 이벤트 시스템

#### 9.1.1 설계 의도

- **개인 서사 심화**: 각 파티원의 배경과 트라우마를 5단계에 걸쳐 점진적으로 공개
- **지역 연계**: 5개 지역 각각에서 다른 측면의 스토리가 전개
- **보상 다양화**: 신뢰도, 고유 카드, 인텐트 해금, 아우로라 기억 파편 등
- **선택적 경험**: 낮은 확률(5~10%)로 발생하여 런마다 다른 경험 제공

#### 9.1.2 발생 조건 및 확률

| 조건 | 기본 확률 | 보정 |
|------|----------|------|
| 해당 파티원 편성 | 5% | +2% (신뢰도 30+), +3% (신뢰도 60+) |
| 해당 지역 노드 진입 | - | 지역별 특정 노드 타입에서만 발생 |
| 이전 단계 완료 | - | 2단계 이후는 이전 단계 완료 필수 |
| 장(Chapter) 진행 | - | 4~5단계는 4장 이후에만 발생 |

**최대 발생 확률**: 10% (신뢰도 60+ 파티원)

#### 9.1.3 서브스토리 단계 구조

| 단계 | 이름 | 신뢰도 요구 | 내용 | 보상 |
|------|------|------------|------|------|
| 1단계 | 과거의 조각 | 10+ | 20년 전 원정대 시절의 단편 회상 | 신뢰도 +5, 파편 조각 10개 |
| 2단계 | 상처의 기억 | 25+ | 검은 재앙 당시 트라우마 직면 | 신뢰도 +8, 기본 고유 카드 |
| 3단계 | 숨겨진 진실 | 40+ | 아우로라와의 관계, 숨겨진 비밀 | 신뢰도 +10, 2번째 인텐트 해금 |
| 4단계 | 용서와 화해 | 55+ | 과거에 대한 결산, 선택 분기 | 신뢰도 +12, 고급 고유 카드 |
| 5단계 | 새로운 맹세 | 70+ | 현재의 결의, 엔딩 영향 | 신뢰도 +15, 전설 카드 + 아우로라 기억 파편 |

#### 9.1.4 지역별 발생 노드 타입

| 지역 | 주요 노드 타입 | 테마 |
|------|---------------|------|
| 안개의 경계 | 이벤트 노드 | 기억의 모호함, 과거와 현재의 혼재 |
| 잿빛 폐허 | 휴식/이벤트 노드 | 상실과 애도, 트라우마 직면 |
| 검은 숲 | 이벤트/전투 노드 | 공포와 결의, 숨겨진 진실 |
| 봉인의 신전 | 신전/스토리 노드 | 아우로라와의 관계, 신앙과 배신 |
| 태초의 던전 | 스토리/보스 노드 | 궁극의 선택, 엔딩 분기 |

#### 9.1.5 파티원 희귀 이벤트 데이터 구조

```
PartyMemberEventData (extends EventData)
├── PartyMemberID         : string     // "PM_ELIAS", "PM_MIRA", etc.
├── PartyMemberName       : string     // 파티원 이름
├── StoryTheme            : string     // 핵심 테마 ("충성과 죄책감", "숨겨진 감정", etc.)
├── StoryPhase            : int        // 1~5
├── Region                : enum       // FogBorder, AshRuins, BlackForest, SealedTemple, PrimordialDungeon
├── TriggerNodeTypes      : array      // [Event, Rest, Temple, Story, Boss]
├── BaseSpawnChance       : float      // 0.05 (5%)
├── TrustBonusThreshold   : array      // [{trust: 30, bonus: 0.02}, {trust: 60, bonus: 0.03}]
├── TrustRequirement      : int        // 단계별 최소 신뢰도
├── ChapterRequirement    : int        // 4~5단계는 4장 이후
├── PreviousPhaseRequired : int        // 2단계 이후는 이전 단계 완료 필수
├── EventTitle            : string     // "부대장의 검", "흐려진 시선", etc.
├── FlavorText            : string     // 이벤트 본문 텍스트
├── Choices               : array      // 선택지 배열 (ChoiceData)
├── Rewards               : object     // 단계별 보상
│   ├── TrustGain         : int        // 신뢰도 증가량
│   ├── UnlockCard        : string     // 해금 카드 ID (nullable)
│   ├── UnlockIntent      : string     // 해금 인텐트 ID (nullable)
│   ├── MemoryFragment    : string     // 아우로라 기억 파편 ID (nullable)
│   └── EndingBranch      : string     // 엔딩 분기 태그 (nullable)
├── RelatedMemoryProgress : int        // 특정 선택지 해금에 필요한 기억 회복 진행도
└── NextPhaseBonus        : float      // 이 이벤트 완료 시 다음 단계 발생 확률 보너스
```

> **파티원별 상세 이벤트**: 노드맵_이벤트_컨텐츠_기획서.md 9절 참조

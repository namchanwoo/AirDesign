## 0. 기획 의도

### 0.1 Why: 왜 세계관에 집중하는가?

**"단순한 덱빌더가 아닌, 런을 하게 만드는 동기부여"**

| 문제 인식 | 세계관의 역할 |
|----------|--------------|
| 로그라이트는 반복 플레이가 핵심 | 반복이 "작업"이 아닌 "탐구"가 되도록 동기 부여 |
| 카드덱빌더의 시스템만으로는 장기 몰입 한계 | 스토리와 캐릭터를 통한 감정적 연결 제공 |
| "왜 다시 던전에 가는가?"에 답 필요 | 주인공의 정체, 파티원의 서사가 답이 됨 |

### 0.2 게임플레이 연결

| 설정 요소 | 게임플레이 연결 |
|----------|----------------|
| **주인공의 권능 (여명)** | 파티원 육성/빌드 시스템의 근거 |
| **파티원 = 원정대원 부활** | 런 반복 시 새 캐릭터 영입/성장 동기 |
| **기억 상실** | 런 진행에 따른 스토리 파편 수집 |
| **검은 재앙** | 던전 난이도 상승, 위협 요소의 근거 |
| **던전 = 봉인의 틈새** | 왜 던전이 존재하는지에 대한 설명 |

### 0.3 컨셉: 중세 판타지 + 비틀기

**"익숙한 중세 판타지 껍데기, 우주적 공포의 속내"**

- 표면: 모험가 길드, 던전 탐험, 영웅 서사
- 이면: SCP/크툴루급 외우주 존재, 인지재해, 신들의 몰락
- 핵심 톤: 이해할 수 없는 것에 대한 공포 + 희망의 가능성

## 1. 세계 구조 개요

### 1.1 3층 세계 구조

**"현실은 태초의 던전 위에, 태초의 던전은 외우주의 경계에"**

| 계층 | 명칭 | 역할 | 게임 등장 |
|:----:|------|------|----------|
| 0 | 외우주 (Outer Void) | 이해 불가능한 존재들의 영역 | 설정상만 존재 |
| 1 | 태초의 던전 (Primordial Dungeon) | 신들의 탄생지, 외우주와의 경계 | 최종 스테이지 |
| 2 | 현실 세계 (아르카니아 대륙) | 도망친 신들이 정착한 중세 판타지 | 메인 무대 |

### 1.2 각 계층 핵심

- **외우주**: 물리법칙/시공간 개념이 다름. 불가해자들의 근원. (레퍼런스: 크툴루 신화, SCP)
- **태초의 던전**: 현재 불완전하게 봉인됨. 주인공의 희생이 봉인의 핵심이었음.
- **현실 세계**: 인간은 신의 피조물이 아닌 독립적 존재. 던전들 = 봉인의 틈새.

## 2. 신들의 역사

### 2.1 역사 연대기

| 시대 | 사건 | 결과 |
|------|------|------|
| **태초** | 태초의 던전에서 신들 탄생 | 주인공(아이온) 포함 |
| **전쟁** | 외우주 존재 침입 + 신들 내전 | 대응 방식 갈등 (싸움/도망/협상) |
| **탈출** | 일부 신들 현실 세계로 도망 | 루(Lux) 포함 |
| **봉인** | 주인공이 봉인의 핵심이 됨 | 스스로 희생을 선택 |
| **현재** | 불완전한 봉인, 틈새로 던전 생성 | 수천 년간 인류가 던전 탐험 |

### 2.2 신들의 분류

| 분류 | 설명 | 대표 |
|------|------|------|
| **남은 자** | 봉인의 일부가 된 신 | 주인공 (아이온) |
| **도망친 자** | 현실 세계로 탈출한 신들 | 신 루 (Lux) |
| **타락한 자** | 외우주에 의해 변질된 신들 | 미정 |

## 3. 검은 재앙

### 3.1 발생 경위 (20년 전)

1. 100명의 영웅으로 구성된 원정대가 태초의 던전 탐험
2. 봉인의 핵심(주인공 시체/유물) 발견 및 운반 결정
3. 봉인력 급격히 약화 -> 외우주의 "침식" 시작
4. 검은 안개가 대륙 중앙 덮음, 원정대 전멸

### 3.2 특성

| 특성 | 내용 | 게임플레이 연결 |
|------|------|----------------|
| **본질** | 현실이 외우주화되는 과정 | 던전 난이도/위협의 근거 |
| **외형** | 검은 안개 형태로 확산 | 시각적 연출 기반 |
| **영향** | 환경 변화, 몬스터 오염, 정신 오염 | 인지오염도 시스템 연동 |
| **시간** | 20년간 지속, 확산 중 | 게임 긴장감 조성 |

## 4. 외우주 존재 계층

| 계층 | 명칭 | 특성 | 게임 역할 |
|:----:|------|------|----------|
| 0 | 불가해자 (Incomprehensible) | 존재 자체 인식 불가 | 설정상만 존재 |
| 1 | 심연자 (Abyssal) | 신들과 전쟁한 존재, 권능급 힘 | 최종 보스 (1~3체) |
| 2 | 현현자 (Manifested) | 자체 영역 형성, 규칙 왜곡 | 중간/지역 보스 |
| 3 | 침투자 (Infiltrators) | 형태 불안정, 본능적 행동 | 일반 적 |

**레퍼런스**: 크툴루 신화 (심연자), SCP 케테르급 (현현자), SCP 유클리드급 (침투자)

## 5. 던전 시스템

### 5.1 던전의 기원

**"던전 = 불완전한 봉인의 틈새"**

- 태초의 던전을 완전히 봉인하지 못해 생긴 틈새에서 형성
- 독특한 아이템, 미지의 몬스터 존재
- 인류가 수천 년간 탐험해온 문화 형성

### 5.2 던전과 인류

| 시대 | 관계 | 게임플레이 연결 |
|------|------|----------------|
| 발견 | 위협으로 인식 | - |
| 탐험 | 모험가 길드 형성 | 길드 시스템 배경 |
| 공존 | 던전 자원 활용, 경제 형성 | 보상/성장 루프 근거 |
| 현재 | 검은 재앙으로 위험도 급상승 | 난이도 상승 근거 |

### 5.3 모험가 길드의 성격 변화

**"탐험가에서 결사대로"**

| 시기 | 성격 | 목적 |
|------|------|------|
| **과거 (검은 재앙 이전)** | 탐험가 | 미지의 세계 탐험, 보물 수집, 명예 추구 |
| **현재 (검은 재앙 이후)** | 결사대 | 검은 재앙 대응, 던전 억제, 인류 생존 |

- 명목상 "모험가"이나, 실질적으로는 **소모되는 전력**
- 빌헬름 도시를 거점으로 검은 재앙 확산을 저지하는 최전선 역할
- 탐험보다 생존과 방어가 우선시되는 분위기

## 6. 인류 문명

### 6.1 컨셉: 중세 판타지 + 비틀기

표면적으로는 일반적인 중세 판타지 세계관을 따르되, 세계의 진실(외우주, 신들의 역사)이 숨겨져 있음.

| 요소 | 표면 | 이면 |
|------|------|------|
| **던전** | 보물과 위험이 있는 미지의 공간 | 외우주와 연결된 봉인의 틈새 |
| **모험가** | 명예와 부를 추구하는 탐험가 | 인류를 외우주 위협에서 지키는 방벽 |
| **신전** | 종교 시설, 부활 서비스 | 도망친 신(루)이 운영 |

### 6.2 주요 요소 (간략)

- **모험가 길드**: 던전 탐험 조율, 등급 관리 (상세는 별도 문서)
- **국가 체계**: 존재하나 게임에서 비중 낮음 (상세는 별도 문서)
- **경제**: 던전 자원 기반 (상세는 별도 문서)

## 7. 주인공 설정

### 7.1 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 아이온 (Aion) - "영원"의 의미 |
| **정체** | 태초의 신 중 하나 |
| **권능** | 여명 (Dawn) |
| **현재** | 기억/능력 대부분 상실, 루의 신전에서 부활 직후 |

### 7.2 권능: 여명 (Dawn)

**"타인의 잠재력/가능성/미래를 열어주는 힘"**

| 측면 | 내용 | 게임플레이 연결 |
|------|------|----------------|
| **본질** | 자신의 미래를 포기한 대신 타인의 미래를 열어줌 | 파티원 육성의 근거 |
| **발현** | 파티원의 잠재력을 끌어올림 | 빌드/강화 시스템 |
| **제약** | 자기 자신에게는 적용 불가 | 주인공은 지원 특화 |

### 7.3 주인공의 과거

1. 태초의 던전에서 탄생, 외우주 존재들과의 전쟁 참여
2. 누군가 남아 봉인이 되어야 했음 -> 자원하여 남음
3. 주인공이 봉인의 핵심이 되기로 자원, 다른 신들은 현실 세계로 이동
4. 수천 년간 시체/유물 형태로 봉인 유지
5. 20년 전 원정대가 발견, 운반 중 전멸, 시체 방치됨
6. 최근 루의 신전에서 의도치 않게 부활

### 7.4 부활 상태

- 시체 상태 불량 (수천 년 + 전투 + 운반 손상) -> 기억/능력 대부분 상실
- "나는 누구인가?"에서 시작 -> 런 진행에 따라 점진적 회복

## 8. 신 루 (Lux) 설정

### 8.1 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 루 (Lux) - "빛/진리"의 의미 |
| **정체** | 도망친 신들 중 하나 |
| **권능** | 진리/부활 |
| **과거** | 아이온과 함께 외우주와 싸웠음, 현실 세계로 이동 |
| **역할** | 현실 세계에서 부활 시스템 운영, 인류 보호 |

### 8.2 부활 시스템

| 요소 | 내용 | 게임플레이 연결 |
|------|------|----------------|
| **비용** | 금(돈) - 인간이 소중히 하는 공통 가치 | 부활 비용 근거 |
| **조건** | 시체 필요, 상태에 따라 결과 상이 | 파티원 초기 능력치 차이 |
| **결과** | 양호->기억/능력 유지, 불량->대부분 상실 | 캐릭터 성장 시스템 |

#### 시체 상태에 따른 부활 결과

**핵심 규칙: 시체 상태 = 기억/능력 보존률**

| 시체 상태 | 기억 보존 | 능력 보존 | 게임플레이 영향 |
|----------|----------|----------|----------------|
| **양호** (손상 최소) | 90% 이상 | 90% 이상 | 거의 온전한 상태로 부활, 즉시 전력화 |
| **보통** (부분 손상) | 50~70% | 50~70% | 일부 기억 단절, 능력 일부 상실 |
| **불량** (심한 손상) | 10~30% | 10~30% | 대부분 상실, 거의 백지 상태에서 시작 |
| **극손상** (수천 년 방치) | 1% 미만 | 1% 미만 | 주인공(아이온)의 경우 - 신의 힘/기억 거의 상실 |

**세계관적 의의**
- 로그라이트 구조를 설정적으로 자연스럽게 설명
- 사망 → 부활 → 능력/기억 변화가 "부활 실패율"로 설명됨
- 반복 플레이가 세계관 내에서 수용됨

### 8.3 루의 신전

- **위치**: 아르카니아 대륙 주요 도시들
- **기능**: 부활 서비스 (유료)
- **게임 역할**: 허브, 부활 포인트, 파티원 영입

## 10. 현재 상황 (게임 시작)

### 10.1 세계 상황

**"검은 재앙 발생 20년 후"**

## 부록. 핵심 설정 빠른 참조

### 3층 세계
1. **외우주**: 이해 불가능한 존재들의 영역 (설정상만)
2. **태초의 던전**: 외우주와 연결된 유일한 공간, 신들 탄생지 (최종 스테이지)
3. **현실 세계**: 도망친 신들이 정착한 아르카니아 대륙 (메인 무대)

### 주인공
- **아이온** (Aion): 태초의 신, 봉인의 핵심이었음, 현재 기억/능력 상실
- **권능 여명**: 타인의 잠재력을 열어줌 -> 파티원 육성 근거

### 신 루
- **루** (Lux): 도망친 신, 현실 세계에서 부활 서비스 운영
- 주인공 부활의 직접적 원인

### 외우주 존재
- 계층 0 불가해자 (미등장) -> 계층 1 심연자 (최종보스) -> 계층 2 현현자 (중간보스) -> 계층 3 침투자 (일반적)

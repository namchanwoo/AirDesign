## 1. 챕터 1 몬스터 개요

### 1.1 설계 방향

**핵심 컨셉**: "검은 재앙에 침식된 존재들"

| 원칙 | 설명 |
|------|------|
| **오염의 시각화** | 모든 몬스터는 검은 재앙의 영향을 받아 변형됨 |
| **과거의 잔재** | 원래는 일반적인 존재였으나, 현재는 왜곡됨 |
| **불완전함** | 형태/정신이 온전하지 않음, 고통과 광기 |
| **초반 난이도** | 게임 첫 본격 전투 지역, 학습 가능한 수준 |

### 1.2 몬스터 분류

| 분류 | 설명 | 예시 |
|------|------|------|
| **침식된 생물** | 원래 생물이 검은 재앙에 오염 | 침식 오크, 오염된 야수 |
| **영적 존재** | 죽은 자의 영혼이 왜곡됨 | 침식 유령 |
| **빈 껍데기** | 물질만 남고 의지는 사라짐 | 갑주 잔해 수호자 |
| **외우주 부산물** | 검은 재앙에서 직접 태어남 | 안개 기생체 |
| **보스** | 챕터 대표 존재 | 타락한 로렌 영주 |

## 2. 기본 몬스터

### 2.1 침식된 오크 (Corrupted Orc)

#### 개요

| 항목 | 내용 |
|------|------|
| **분류** | 침식된 생물 |
| **원래 존재** | 부족 단위로 살던 오크 |
| **변화** | 검은 연기에 노출되어 광폭화 |
| **역할** | 챕터 1의 기본 전투 몬스터 |

#### 외형 특징
- 무기와 살점이 뒤틀림
- 검은 연기가 피부 틈에서 새어나옴
- 눈이 검게 변했거나, 완전히 없어짐
- 과도하게 비대해진 근육, 불균형한 체형

#### 디자인 키워드
- 광폭화
- 뒤틀림
- 본능 폭주
- 이성 상실

#### 인텐트 패턴 (예시)

| 인텐트 | 실행 조건 | 행동 |
|--------|----------|------|
| 돌진 | 전방 적 존재 시 | 전방 1칸 이동 + 근접 공격 |
| 광란 | HP 50% 이하 | 공격력 증가 + 랜덤 공격 |
| 분노의 포효 | 동료 사망 시 | 자신 버프 + 주변 아군 버프 |

#### 출현 지역
- 검은 숲 전역
- 로렌 가도
- 폐허 마을 외곽

### 2.2 침식 유령 (Corrupted Specter)

#### 개요

| 항목 | 내용 |
|------|------|
| **분류** | 영적 존재 |
| **원래 존재** | 로렌 영토 주민 혹은 병사의 영혼 |
| **변화** | 평범한 유령이 아닌, 형태가 불완전하고 기괴함 |
| **역할** | 심리적 압박, 디버프 특화 |

#### 외형 특징
- 형태가 불완전함 (반쯤 사라진 모습)
- 얼굴, 팔다리가 비정상적으로 늘어남
- 고통에 일그러진 표정이 고정됨
- 반투명, 흐릿한 경계

#### 디자인 키워드
- 공포
- 비틀림
- 고통의 잔상
- 불완전한 존재

#### 인텐트 패턴 (예시)

| 인텐트 | 실행 조건 | 행동 |
|--------|----------|------|
| 한탄 | 기본 | 적 1명에게 정신 피해 + 디버프 |
| 망령의 손길 | 인접 적 존재 시 | 빙의 시도 (행동 방해) |
| 분산 | HP 낮을 시 | 여러 조각으로 분리 (분열) |

#### 출현 지역
- 폐허 마을
- 오염된 하천
- 로렌 성채 내부

### 2.3 갑주 잔해 수호자 (Hollow Armor Guardian)

#### 개요

| 항목 | 내용 |
|------|------|
| **분류** | 빈 껍데기 |
| **원래 존재** | 로렌 성채 병사들의 갑옷 |
| **변화** | 내부는 비어 있음, 검은 기운이 갑옷을 움직임 |
| **역할** | 방어형, 지연 전술 |

#### 외형 특징
- 중세 석조 성채의 수비병 갑옷
- 내부가 비어 있음 (투구 안이 어둠)
- 관절 부분에서 검은 연기가 새어나옴
- 녹슬고 손상된 부분이 있으나 움직임

#### 디자인 키워드
- 빈 껍질
- 자동 인형
- 무기화된 잔재
- 기계적 움직임

#### 인텐트 패턴 (예시)

| 인텐트 | 실행 조건 | 행동 |
|--------|----------|------|
| 수호 | 기본 | 방어 자세 (피해 감소) |
| 순찰 | 적 미발견 시 | 정해진 경로 이동 |
| 반격 | 피격 시 | 근접 반격 |
| 분해 | 파괴 시 | 파편 비산 (주변 피해) |

#### 출현 지역
- 로렌 성채 외벽
- 로렌 성채 내부
- 로렌 가도 (성채 근처)

### 2.4 오염된 야수 (Corrupted Beast)

#### 개요

| 항목 | 내용 |
|------|------|
| **분류** | 침식된 생물 |
| **원래 존재** | 숲의 수호 동물 (늑대, 곰, 사슴 계열) |
| **변화** | 신체 일부가 과도하게 증식, 본능 폭주 |
| **역할** | 기동성, 기습 공격 |

#### 외형 특징
- 과도하게 증식된 신체 일부 (이빨, 발톱, 뿔)
- 눈이 여러 개로 증식되거나, 완전히 사라짐
- 털이 검게 변하거나, 피부가 드러남
- 기형적인 체형 (비대칭)

#### 디자인 키워드
- 자연의 붕괴
- 과잉 진화
- 본능 폭주
- 비정상적 생명력

#### 인텐트 패턴 (예시)

| 인텐트 | 실행 조건 | 행동 |
|--------|----------|------|
| 사냥 | 기본 | 약한 적 추적 + 공격 |
| 도약 | 거리 2칸 이상 | 점프 공격 (거리 무시) |
| 포식 | 적 처치 시 | HP 회복 |
| 울부짖음 | HP 낮을 시 | 동료 소환 |

#### 출현 지역
- 검은 숲 전역
- 오염된 하천 주변

### 2.5 안개 기생체 (Mist Parasite)

#### 개요

| 항목 | 내용 |
|------|------|
| **분류** | 외우주 부산물 |
| **원래 존재** | 없음 - 검은 연기에서 직접 태어남 |
| **특성** | 형태가 일정하지 않음, 실체와 비실체의 중간 |
| **역할** | 특수 몬스터, 디버프/침식 |

#### 외형 특징
- 검은 안개가 응집된 형태
- 형태가 계속 변함 (불안정)
- 촉수나 가시가 간간이 나타났다 사라짐
- 주변 빛을 흡수하는 느낌

#### 디자인 키워드
- 흐릿함
- 존재 불명확
- 외우주의 부산물
- 실체 없는 위협

#### 인텐트 패턴 (예시)

| 인텐트 | 실행 조건 | 행동 |
|--------|----------|------|
| 침식 | 기본 | 인접 적에게 지속 피해 |
| 흡수 | 인접 적 존재 시 | 적 버프 제거 + 자신 강화 |
| 분산 | 피격 시 | 확률적 피해 회피 |
| 융합 | 동료 기생체 인접 시 | 합체하여 강화 |

#### 출현 지역
- 오염된 하천
- 검은 숲 심층부
- 안개가 짙은 지역 전반

## 3. 챕터 1 보스: 타락한 로렌 영주

### 3.1 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 타락한 로렌 영주 (Corrupted Lord Loren) |
| **원래 존재** | 로렌 영토의 영주, 원정대 100인 중 한 명 (80~100위권) |
| **변화** | 죄책감과 공포에 잠식, 검은 재앙과 동화 |
| **위치** | 로렌 성채 내부 최심부 |

### 3.2 서사적 역할

| 상징 | 설명 |
|------|------|
| **도망쳤으나 살아남은 자** | 원정대에서 철수 후 성채에 남음 |
| **진실을 알지 못한 증인** | 일부만 알고, 핵심은 모름 |
| **첫 번째 고백자** | 플레이어에게 불완전한 진실의 첫 조각 제공 |

### 3.3 외형 특징

**1단계 (전투 시작)**
- 과거 영주의 면모가 남아있음
- 갑옷은 훼손되었으나 형태 유지
- 검은 기운이 갑옷 틈에서 새어나옴
- 표정에 고통과 죄책감

**2단계 (HP 50% 이하)**
- 검은 재앙과 완전히 동화
- 갑옷이 살점과 융합
- 비대해진 체형, 괴물에 가까움
- 광기에 휩싸인 모습

**3단계 (패배 직전)**
- 인간의 모습 일부 회복
- 정신이 잠시 돌아옴
- 고백 가능한 상태

### 3.4 디자인 키워드

- 죄책감
- 자기 처벌
- 동화
- 괴물화
- 순간의 인간성

### 3.5 인텐트 패턴 (예시)

**1단계 패턴**

| 인텐트 | 행동 |
|--------|------|
| 영주의 일격 | 강력한 단일 공격 |
| 방어 명령 | 주변 갑주 잔해 소환 |
| 참회 | 자해 + 공격력 증가 |

**2단계 패턴**

| 인텐트 | 행동 |
|--------|------|
| 검은 폭풍 | 광역 공격 |
| 포식 | 아군 몬스터 흡수 + 회복 |
| 광란 | 연속 공격 (랜덤 대상) |

**3단계 패턴**

| 인텐트 | 행동 |
|--------|------|
| 마지막 저항 | 전력 공격 |
| 붕괴 | 자폭 시도 (방해 가능) |

### 3.6 전투 후 이벤트

보스 처치 후, 로렌 영주는 잠시 제정신을 차리고 고백한다:

> "우리는 정복하러 갔다"
> "그곳엔 우리가 감당할 수 없는 것이 있었다"
> "그리고... 네가 있었다"
> "우리는 봉인된 걸 몰랐다"
> "우린... 망쳤다"
> "난 시체를 운반했을 뿐이다"

→ 불완전한 진실, 더 중요한 인물들(엘리아스 등)은 2장에서 등장
